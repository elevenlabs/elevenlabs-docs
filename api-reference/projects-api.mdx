## 1. Introduction

The Projects API is a powerful tool for creating voiceovers for long-form content such as articles, blog posts, or audiobooks. It allows you to generate audio from text, and then stitch together the audio clips to create a seamless narration. The API is designed to be easy to use, and provides a number of features to help you create high-quality voiceovers.

**Note: The Projects API is currently only available on our [enterprise](https://elevenlabs.io/enterprise) plan**.
