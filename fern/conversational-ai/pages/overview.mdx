## What is Conversational AI?

ElevenLabs [Conversational AI](https://elevenlabs.io/conversational-ai) is a platform for deploying customized, conversational voice agents. Built in response to our customers' needs, our platform eliminates months of development time typically spent building conversation stacks from scratch. It combines these building blocks:

<CardGroup cols={2}>

  </Card>
  <Card title="Text to speech">

  </Card>
  <Card title="Turn taking">
    Our custom turn taking and interruption detection service that feels human.

Altogether it is a highly composable AI Voice agent solution that can scale to thousands of calls per day. With [server](/docs/conversational-ai/customization/tools) & [client side](/docs/conversational-ai/customization/client-tools) tools, [knowledge](/docs/conversational-ai/customization/knowledge-base) bases, [dynamic](/docs/conversational-ai/customization/dynamic-variables) agent instantiation and [overrides](/docs/conversational-ai/customization/overrides), plus built-in monitoring, it's the complete developer toolkit.

<script>
  {`
  const tierData = {
    creator: { base: 22, included: 250, rate: 0.12 },
    pro: { base: 99, included: 1100, rate: 0.11 },
    scale: { base: 330, included: 3600, rate: 0.10 },
    business: { base: 1320, included: 13750, rate: 0.08 }
  };

  function formatNumber(num) {
    return new Intl.NumberFormat().format(num);
  }

  function updateCalculator() {
    const tierSelect = document.getElementById('pricingTier');
    const minutesSlider = document.getElementById('minutesSlider');
    const minutesValue = document.getElementById('minutesValue');
    const totalCost = document.getElementById('totalCost');
    
    if (!tierSelect || !minutesSlider || !minutesValue || !totalCost) return;

    const tier = tierSelect.value;
    const minutes = parseInt(minutesSlider.value);
    const tierInfo = tierData[tier];
    
    minutesValue.textContent = formatNumber(minutes) + ' minutes';
    
    let cost = tierInfo.base;
    if (minutes > tierInfo.included) {
      cost += (minutes - tierInfo.included) * tierInfo.rate;
    }
    
    totalCost.textContent = '$' + formatNumber(Math.round(cost));
  }

  function updateSliderValue(slider) {
    const sliderValue = document.getElementById('sliderValue');
    const value = parseInt(slider.value);
    const percent = ((value - slider.min) / (slider.max - slider.min)) * 100;
    
    // Update the floating value position
    sliderValue.style.left = \`\${percent}%\`;
    sliderValue.textContent = formatNumber(value);
  }

  function showSliderValue() {
    const sliderValue = document.getElementById('sliderValue');
    sliderValue.style.opacity = '1';
  }

  function hideSliderValue() {
    const sliderValue = document.getElementById('sliderValue');
    sliderValue.style.opacity = '0';
  }

  document.addEventListener('DOMContentLoaded', function() {
    const tierSelect = document.getElementById('pricingTier');
    const minutesSlider = document.getElementById('minutesSlider');
    
    if (tierSelect) {
      tierSelect.addEventListener('change', updateCalculator);
    }
    
    if (minutesSlider) {
      minutesSlider.addEventListener('input', function() {
        updateCalculator();
        updateSliderValue(this);
      });
      minutesSlider.addEventListener('mouseenter', showSliderValue);
      minutesSlider.addEventListener('mouseleave', hideSliderValue);
      
      // Initialize slider value
      updateSliderValue(minutesSlider);
    }
  });
`}
</script>

## Popular applications

Companies and creators use our Conversational AI orchestration platform to create:

- **Customer service**: Assistants trained on company documentation that can handle customer queries, troubleshoot issues, and provide 24/7 support in multiple languages.
- **Virtual assistants**: Assistants trained to manage scheduling, set reminders, look up information, and help users stay organized throughout their day.
- **Retail support**: Assistants that help customers find products, provide personalized recommendations, track orders, and answer product-specific questions.
- **Personalized learning**: Assistants that help students learn new topics & enhance reading comprehension by speaking with books and [articles](https://elevenlabs.io/blog/time-brings-conversational-ai-to-journalism).
